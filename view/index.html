¿Qué es HTML5?

Esto fue posteado el 14 de agosto de 2024 por UMG-Guastatoya

HTML5 es la última evolución de HTML, el estándar básico de la web. HTML5 se diseñó para ser utilizable en todos los dispositivos, desde PC a teléfonos móviles pasando por tabletas, y también para ser legible por humanos y entendible por las computadoras y otros dispositivos de análisis. HTML5 incluye las siguientes características:

  • Un lenguaje de marcado más versátil para la web
  • Nuevos elementos estructurales
  • Nuevos atributos
  • Funciones de edición de contenido
  • Arrastrar y soltar
  • APIs de audio y vídeo
  • Elementos gráficos 2D y 3D
  • Almacenamiento local en el lado del cliente
  • Comunicaciones asíncronas
  • APIs de dispositivos
  • Y mucho más...

¿Qué es CSS3?

Esto fue posteado el 14 de agosto de 2024 por UMG-Guastatoya

CSS, o Cascading Style Sheets, es un lenguaje que se utiliza para describir la presentación de una página web. Mientras que HTML se encarga de la estructura y el contenido de la página, CSS define cómo se verá ese contenido: los colores, las fuentes, los márgenes, el diseño y la disposición de los elementos en la página.

CSS3 es la última evolución de CSS, y añade muchas características nuevas y emocionantes a las hojas de estilo. Algunas de las nuevas características de CSS3 incluyen:

  • Soporte para fuentes web
  • Colores RGBA y HSLA
  • Gradientes
  • Transiciones y animaciones
  • Transformaciones 2D y 3D

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para crear y controlar el comportamiento dinámico de las páginas web. A diferencia de HTML y CSS, que se encargan de la estructura y el estilo, JavaScript permite que las páginas web respondan a las acciones del usuario, como hacer clic en un botón, rellenar formularios, o cargar nuevos datos sin recargar la página.
